Tag Loader

Aplikasi ini akan meload foto dengan tag yang diberikan. Kami hanya bisa meload 1 tag untuk sekarang ini dan hanya gambar yang dapat kami load. Aplikasi ini tidak akan mendownload media berupa video, dan membuat zip file dari gambar gambar yang telah terdownload.

Berdasarkan peraturan yang diberikan Instagram, anda hanya dapat menggunakan 500 request per jam untuk ID normal, dan 5000 request per jam untuk ID yang sudah terverifikasi. mohon bersabar karena kami tidak dapat melakukan kedua hal tersebut dengan mudah. Ini akan berlangsung mudah.
